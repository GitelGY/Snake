🐍 Snake Concerto - Full Version

משחק נחש מודרני, מהיר ועשיר בתכונות שנבנה ב-HTML5 Canvas ו-JavaScript. המשחק מציע חוויית משתמש משופרת עם מערכת בונוסים, בחירת עיצובים וטבלת שיאים מקומית.

✨ תכונות עיקריות
מערכת התחברות: הזנת שם שחקן ומעקב אישי.

רמות קושי: בחירה בין קל, רגיל וקשה (משפיע על מהירות הנחש).

עיצובים מותאמים (Themes): שלושה מצבי תצוגה - לילה (Dark Mode), וינטאז' ויער.

התאמה אישית: בחירה מתוך מגוון צבעים עבור הנחש.

מכשולים: קירות אקראיים הנוצרים בכל משחק ומאתגרים את השחקן.

טבלת שיאים (Leaderboard): שמירה אוטומטית של 5 התוצאות הטובות ביותר ב-LocalStorage.

🍎 מכניקת הבונוסים
מעבר לתפוח האדום הרגיל, המשחק כולל "פירות" מיוחדים שמופיעים לזמן מוגבל:

סוג האוכל	צבע	השפעה
תפוח רגיל	אדום	נקודה אחת וגדילה של הנחש.
תפוח זהב	צהוב	מעניק 5 או 10 נקודות בונוס מיידיות.
תפוח קסם	כחול	מקצר את הנחש ומפעיל מצב X2 ל-10 שניות!
🎮 איך משחקים?
תנועה: השתמשו במקשי החיצים במקלדת כדי לנווט את הנחש.

הפסקה: לחצו על מקש ה-Space כדי לעצור את המשחק בכל שלב.

פסילה: נגיעה בקירות, בגוף הנחש או במכשולים תסיים את המשחק.

🛠️ טכנולוגיות
Vanilla JavaScript (ES6+)

HTML5 Canvas לרינדור הגרפיקה.

Web Audio API ליצירת צלילים דינמיים ללא קבצי שמע חיצוניים.

CSS Variables לניהול ערכות הנושא.
